Dynamic Scroll

@BenjiWatkins - Github

A JS script (with CSS) to change the colour of your scrollbar as you scroll!

Link the neccessary files (dynamicScroll.js, dynamicScroll.css, and JQuery)

<link rel="stylesheet" href="dynamicScroll.css">

<script src="query-1.11.2.min.js"> </script> <script src="dynamicScroll.js"> </script>

Then in the body onload function, run dynamicfy(r1, g1, b1, r2, g2, b2) where the r g and b values are rgb colour codes to fade between, as shown in the example below

<body onload="dynamicfy(255,100,150,100,250,250);">

Which fades between rgb(255,100,150) and rgb(100,250,250), and can be seen on the example page

http://dynamicscroll.tumblr.com/

The smooth effect works by calculating the difference in rgb values, dividing it by 30 (a changeable number of steps) and then when you scroll, it works out which interval you need! If that was a rubbish explanation try it yourself or have a look at the source code!
